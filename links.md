# Links

## Windows

Cria uma "junção" de pastas que sincroniza com o GoogleDrive.
O FolderOrigem não deve existir, e o FolderOrigem destino deve ser uma pasta criada.

```terminal
  mklink /J D:<Path>\<FolderOrigem> D:<Path>\<FolderDestino>
```
